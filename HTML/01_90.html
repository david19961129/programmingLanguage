HTML 格式規範

  • 必須要有 <DOCTYPE>
  • 必須要有 <html>、<head>、<body>
  • HTML 副檔名使用 .html、檔案名稱使用小寫字母
  • 替網頁加上語系與編碼規則、使用空格、換行和特殊符號需要注意
  • 注意結尾標籤、標籤和屬性使用小寫字母
  • 屬性的值加上引號、等號前後不要有空白
  • 加上換行與縮排、空元素後方的斜線

常見tag

項目 說明 範例
<html lang="en"> 指定網頁的語言為英文。 <html lang="en">...</html>
<head> 包含網頁的元數據，如標題、樣式、字元編碼等。 <head>...</head>
<meta charset="UTF-8"> 指定網頁的字元編碼，UTF-8 是最常用的編碼方式。 <meta charset="UTF-8">
<title> 設定網頁的標題，顯示在瀏覽器的標籤上。 <title>My Web Page</title>
<body> 包含網頁的主要內容，如文本、圖片、表格等。 <body>...</body>
<script> 用來嵌入或引用 JavaScript 代碼。 <script>...</script>
<link rel="stylesheet" href="styles.css"> 連結外部樣式表，來設定網頁的樣式。 <link rel="stylesheet" href="styles.css">
<meta name="viewport" content="width=device-width, initial-scale=1.0"> 設定視窗的顯示比例，通常用來適配移動裝置。 <meta name="viewport" content="width=device-width, initial-scale=1.0">
